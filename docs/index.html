Cargo Manifest

9 crates in the container yard. Modular Rust workspace, each crate ships independently.

api-server

Axum-based HTTPS REST API. Routes, handlers, auth, watch, webhooks.

controller-manager

31 reconciliation controllers: deployments, jobs, endpoints, GC, TTL, HPA, and more.

scheduler

Pod scheduling with filter/score plugins, affinity, taints, priority, preemption.

kubelet

Node agent. Container lifecycle via bollard (Docker API), volumes, probes, networking.

kube-proxy

Service routing via iptables. ClusterIP, NodePort, LoadBalancer. Host network mode.

storage

Storage trait with etcd and in-memory backends. Resource versioning via mod_revision.

common

Shared resource types (Pod, Service, Deployment, etc.), error handling, utilities.

kubectl

CLI tool for cluster interaction. get, create, apply, delete, logs, exec, and more.

cloud-providers

AWS, GCP, and Azure integrations for LoadBalancer and cloud-native features.

Quick Start

Get a local cluster running in minutes with Docker Compose.

terminal 
# Clone and build
git clone https://github.com/calfonso/rusternetes.git
cd rusternetes
cargo build --release

# Start the cluster with Docker Compose
export KUBELET_VOLUMES_PATH=$(pwd)/.rusternetes/volumes
docker compose build
docker compose up -d

# Bootstrap CoreDNS, services, and SA tokens
bash scripts/bootstrap-cluster.sh

# Use the cluster
export KUBECONFIG=~/.kube/rusternetes-config
kubectl get nodes
kubectl get pods -A

Why Rust?

Memory Safety

No garbage collector, no null pointer dereferences, no data races. Rust's ownership model eliminates entire classes of bugs that plague C/C++ and even Go control planes.

Performance

Zero-cost abstractions and no runtime overhead. Async I/O via Tokio delivers high throughput with minimal resource consumption compared to goroutine-based implementations.

Correctness

The type system catches mismatches at compile time. Serde ensures wire-compatible JSON serialization. 1,271+ commits and 133 rounds of conformance testing validate behavior against real Kubernetes.
